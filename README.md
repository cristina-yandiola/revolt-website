# Revolt - Astro Theme

This is an Astro website template built with Tailwind CSS.

## Prerequisites

- Node.js version 22.12.0 or later.
- `npm` (comes with Node.js)

## Step-by-Step Setup

1. **Ensure you have the correct Node.js version:**

   ```bash
   nvm use 22.12.0
   ```

   _(If you don't use nvm, just ensure your node version is >= 22.12.0)_

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   The site will be available locally at `http://localhost:4321`.

## Production Build

To build the project for production, run:

```bash
npm run build
```

This will generate the optimized static files in the `./dist/` directory.

To preview the production build locally before deploying:

```bash
npm run preview
```
